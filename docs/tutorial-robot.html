Tutorial: Crear un robot

Guía rápida para definir un nuevo robot usando la estructura de robots/. Puedes copiar robots/darth_vader o robots/charro_negro como plantilla.

1. Estructura de carpetas

Crea una carpeta dentro de robots/, por ejemplo mi_robot/, con:

  • config.json — Configuración del robot (IA, voz, banderas, idle, etc.).
  • secrets.json — Credenciales (p. ej. google_api_key). No subas este archivo al repositorio.
  • ComponentNameDefinition.py — Enums de componentes (servos, LEDs, audios) que heredan de ComponentEnum.
  • SequenceNameDefinition.py — Diccionario nombre → { description, sequence } para animaciones.
  • sequences/ — Módulos Python que definen listas de movimientos (una lista por animación).
  • audios/ — Archivos WAV para idle y secuencias (opcional).
  • main.py — Punto de entrada: carga config, instancia todos los módulos, asigna parámetros y llama a main.start().

2. config.json — Significado de cada campo

La siguiente tabla describe los campos del config.json usando como referencia el de robots/darth_vader/config.json.

Campo Significado
ai.instructions Texto de instrucciones del sistema para la IA: personalidad, reglas de la conversación y formato de respuesta (p. ej. una bandera por frase). Al final se concatenan automáticamente las descripciones de las animaciones disponibles (salvo las excluidas).
ai.first_model_message Primer mensaje que dirá el robot al iniciar la conversación. Debe incluir banderas de animación si quieres gestos (ej. [ADEMAN_1] Soy Darth Vader...).
ai.model Nombre del modelo de IA (p. ej. gemma-3-27b-it para Google AI).
finalization_flags Lista de cadenas. Si alguna aparece en la respuesta de la IA, se termina el bucle conversacional (ej. ["FIN ACEPTA", "FIN ATACA", "FIN"]).
robot_name Nombre del robot (logs y mensajes).
vosk_model_path Ruta al directorio del modelo Vosk para reconocimiento de voz (se usa si instancias VoskSpeechRecognition).
audio_filter.pitch_semitones Desplazamiento de tono en semitonos aplicado al audio de voz (negativo = más grave).
audio_filter.echo_delay Retardo del eco en segundos (ej. 0.008).
audio_filter.echo_decay Decaimiento del eco (0–1).
audio_filter.overlay_delay Retardo de superposición (mezcla).
audio_filter.gain Ganancia del audio (volumen, ej. 0.5 o 1.0).
voice.engine Identificador del motor de voz (ej. "kokoro").
voice.onnx_file_path Ruta al archivo ONNX del modelo Kokoro.
voice.voices_file_path Ruta al archivo de voces (.bin) de Kokoro.
voice.voice Nombre de la voz (ej. pm_alex).
voice.language Código de idioma (ej. es).
voice.speed Velocidad de habla (1.0 = normal).
speech_replacements Diccionario palabra → reemplazo para el sintetizador (pronunciación). Se usa con WordReplacerWrapper (ej. "Darth": "Dart").
idle.sound_wavs Lista de rutas relativas a la carpeta del robot (ej. ["/audios/breathing.wav"]) para sonidos que se reproducen en bucle durante el idle (según idle_behaviour en main.py).
idle.play_audio_interval Intervalo en segundos entre reproducciones de un sonido aleatorio del idle (ej. 3.0).
sequence_audios Rutas de WAV usados en secuencias (referencia; en Darth Vader los audios se cargan desde ComponentNameDefinition.AudioNames con createAudios).
excluded_sequences_from_ai_prompt Nombres de animaciones que no se añaden al prompt de la IA (ej. IDLE, FIN ACEPTA), para que la IA no las elija como gestos normales.
idle_sequence Nombre de la secuencia que se ejecuta cuando el robot está en espera (ej. "IDLE"). Debe existir en SequenceNameDefinition.

3. Definir componentes (ComponentNameDefinition.py)

Cada enum representa un conjunto de componentes manejados por un mismo controlador. Sintaxis por miembro:

  • Servos/LEDs: NOMBRE = (channel, min_value, max_value)
  • Audios (para AudioPlayerController): NOMBRE = (channel, 0, 1, ruta_wav) — el cuarto valor es el label (ruta del archivo).

Ejemplo (Darth Vader):

class Servos(ComponentEnum):
    NECK_X = (1, 0, 180)
    SHOULDER_L_X = (3, 0, 180)
    ...

class Leds(ComponentEnum):
    EYES = (0, 0, 255)
    LIGHT_SABER = (1, 0, 255)

class AudioNames(ComponentEnum):
    SABER_ON = (0, 0, 1, f"{BASE_DIR}/audios/saber_on.wav")
    ...

4. Definir secuencias (sequences/)

Cada archivo en sequences/ (excepto __init__.py) puede exportar listas. El __init__.py descubre los módulos y rellena el diccionario SEQUENCES (nombre de la variable → lista).

Cada secuencia es una lista de pasos. Cada paso es un dict:

  • "parameters": { "time": segundos, "type": "linear" }
  • "components": { ComponentEnum: valor_numérico o None } — Si el valor es None, se aplica de inmediato.

Ejemplo (Ademan_1.py):

from ..ComponentNameDefinition import *

Ademan_1 = [
  { "parameters": { "time": 0.9, "type": "linear" },
    "components": {
      Servos.NECK_X : 0,
      Servos.SHOULDER_L_X : 0,
      ...
    },
  },
  ...
]

5. SequenceNameDefinition.py

Asocia un nombre (el que la IA usará en las banderas) con descripción (para el prompt) y la secuencia (desde SEQUENCES):

from .sequences import SEQUENCES

SequenceNameDefinition = {
    "ADEMAN_1": {
        "description": "Ademán simple mientras habla 1",
        "sequence": SEQUENCES.get("Ademan_1")
    },
    "IDLE": { "description": "No hace nada", "sequence": SEQUENCES.get("Idle") },
    "FIN ACEPTA": { "description": "Animación de término cuando acepta", "sequence": SEQUENCES.get("SoundEffect") },
    ...
}

6. main.py — Cómo inicializar e instanciar cada módulo

Orden típico con fragmentos de código concretos (basados en robots/darth_vader/main.py).

6.1 Base y configuración

from pathlib import Path
import json

BASE_DIR = Path(__file__).resolve().parent

with open(f'{BASE_DIR}/config.json', 'r', encoding='utf-8') as f:
    config = json.load(f)
with open(f'{BASE_DIR}/secrets.json', 'r') as f:
    secrets = json.load(f)

6.2 IA

Crear el modelo, leer instrucciones y primer mensaje, añadir al prompt las descripciones de las animaciones (excluyendo las de excluded_sequences_from_ai_prompt), y fijar el mensaje inicial.

from ai_talking_robot.ai.GoogleAIModel import GoogleAIModel
from .SequenceNameDefinition import SequenceNameDefinition

_ai_model_name = config['ai'].get('model')
ai = GoogleAIModel(apiKey=secrets["google_api_key"], model=_ai_model_name)

_ai_instructions = config['ai']['instructions']
_ai_firstMessage = config['ai']['first_model_message']

_excluded_sequences = config.get('excluded_sequences_from_ai_prompt', [])
for name, d in SequenceNameDefinition.items():
    if d.get("description") is not None and name not in _excluded_sequences:
        _ai_instructions += f"[{name}]: {d['description']}\n"

ai.addMessage(_ai_instructions, _ai_firstMessage)

6.3 Reconocimiento de voz

from ai_talking_robot.voice.DummyTextRecognition import DummyTextRecognition
from ai_talking_robot.voice.VoskSpeechRecognition import VoskSpeechRecognition

# Pruebas sin micrófono:
recognition = DummyTextRecognition()
# Con micrófono (Vosk):
# recognition = VoskSpeechRecognition(model=config.get('vosk_model_path', 'resources/models/vosk-model-es-0.42'))

6.4 Fábrica de audio y filtro

from ai_talking_robot.audioplayer.SimpleAudioFactory import SimpleAudioFactory
from ai_talking_robot.audioplayer.AudioFactoryFilter import AudioFactoryFilter

audioFactory = SimpleAudioFactory()
audioFilter = AudioFactoryFilter(audioFactory)

_audio_filter_config = config.get('audio_filter', {})
audioFilter.pitch_semitones = _audio_filter_config.get('pitch_semitones', -4)
audioFilter.echo_delay = _audio_filter_config.get('echo_delay', 0.008)
audioFilter.echo_decay = _audio_filter_config.get('echo_decay', 0.6)
audioFilter.overlay_delay = _audio_filter_config.get('overlay_delay', 0.01)
audioFilter.gain = _audio_filter_config.get('gain', 1.0)

6.5 Sintetizador de voz

from ai_talking_robot.speech.KokoroSynthetizer import KokoroSynthetizer
from ai_talking_robot.speech.WordReplacerWrapper import WordReplacerWrapper

_voice_config = config.get('voice', {})
synth = KokoroSynthetizer(
    audioFilter,
    onnxFilePath=_voice_config.get('onnx_file_path', 'resources/models/kokoro/kokoro-v1.0.onnx'),
    voicesBinPath=_voice_config.get('voices_file_path', 'resources/models/kokoro/voices-v1.0.bin')
)
synth.setConfig(
    voice=_voice_config.get('voice', 'pm_alex'),
    language=_voice_config.get('language', 'es'),
    speed=_voice_config.get('speed', 1.0)
)
synth = WordReplacerWrapper(synth, config.get('speech_replacements', {}))

6.6 Activador de entrada y controladores

from ai_talking_robot.input.EnterInputActivator import EnterInputActivator
from ai_talking_robot.controllers.MockController import MockController
from ai_talking_robot.controllers.RemoteUDPController import RemoteUDPController
from ai_talking_robot.controllers.AudioPlayerController import AudioPlayerController
from ai_talking_robot.audioplayer.AudioSpec import WavFileSpec
from .ComponentNameDefinition import Servos, Leds, AudioNames

input_activator = EnterInputActivator()

udp_format = { "servo": "name", "angle": "value" }
servoController = RemoteUDPController(Servos, udp_format, "127.0.0.1", 5005)
# servoController = MockController(Servos, "Servos")  # alternativa sin hardware

ledController = MockController(Leds, "Leds")

audioController = AudioPlayerController(AudioNames, None)
audioController.createAudios(audioFactory, WavFileSpec(""))

6.7 Secuenciador y director de diálogo

from ai_talking_robot.sequencer.DefaultMoveSequencer import DefaultMoveSequencer
from ai_talking_robot.integration.DialogueDirector import DialogueDirector

sequencer = DefaultMoveSequencer()
director = DialogueDirector(synth, sequencer, SequenceNameDefinition)

6.8 Robot y configuración final

from ai_talking_robot.integration.Robot import Robot, Status
import random
import time

main = Robot(
    aiModel=ai,
    recognition=recognition,
    sequencer=sequencer,
    inputA=input_activator,
    director=director,
    animations=SequenceNameDefinition
)

main.robotName = config.get('robot_name', 'Robot')
main.initial_dialogue_text = _ai_firstMessage
main.end_dialogue_flags = config.get('finalization_flags')
main.idle_animation_name = config.get("idle_sequence")

# Comportamiento idle opcional: sonidos en bucle
_idle_cfg = config.get('idle', {})
_audio_wavs = _idle_cfg.get('sound_wavs', [])
_audios = [audioFactory.create(WavFileSpec(path=f'{BASE_DIR}{p}')) for p in _audio_wavs]
_play_audio_interval = _idle_cfg.get('play_audio_interval', 6.0)

def sound_loop():
    while True:
        if main.status in [Status.LISTENING, Status.THINKING, Status.WAITING]:
            if _audios:
                _audios[random.randint(0, len(_audios) - 1)].playAudio()
        time.sleep(_play_audio_interval)

main.idle_behaviour = sound_loop
main.start()

7. Qué controlador usar

Según tu hardware: MockController (pruebas), ArduinoSerialController (Arduino), AdafruitServoController (Raspberry + PCA9685), LedController (GPIO), RemoteUDPController (red). Para sonidos en secuencias, AudioPlayerController. Ver Controladores para instanciar cada uno.

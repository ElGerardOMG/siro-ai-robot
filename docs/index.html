AI Talking Robot

Documentación del proyecto en español.

¿Qué es este proyecto?

Es un framework para programar un robot físico con capacidad de conversar con personas. Combina:

  • Síntesis de voz — para que el robot hable (por defecto Kokoro TTS).
  • Reconocimiento de voz — para escuchar al usuario (por defecto Vosk).
  • Inteligencia artificial — para generar respuestas (por defecto Google AI).
  • Control físico — servos, LEDs, audio, etc., mediante controladores intercambiables.

El diseño permite cambiar cada tecnología (otro TTS, otro modelo de IA, otro reconocedor) sin reescribir el núcleo del código, gracias a clases abstractas (prefijo A) e implementaciones concretas.

¿Cómo funciona en una frase?

El programa principal (Robot.py) ejecuta un bucle conversacional: escucha al usuario, envía el texto a la IA y reproduce la respuesta del robot. En el prompt se le pide a la IA que formatee sus respuestas con banderas entre corchetes (por ejemplo [ADEMAN_1] o [FIN ACEPTA]). Esas banderas disparan animaciones o secuencias en el robot (movimientos, sonidos) o indican el fin de la conversación.

Estructura del repositorio

  • src/ai_talking_robot/ — Código principal (integración, controladores, secuenciador, voz, IA, audio, entrada).
  • robots/ — Implementaciones de ejemplo: Charro Negro y Darth Vader, listas para ejecutar.
